# Internet Speed Test

CLI-утилита для измерения средней скорости скачивания файла по HTTP.

Скрипт:

* читает URL из конфигурационного файла;
* выполняет последовательные HTTP-запросы;
* измеряет время выполнения каждого запроса;
* вычисляет общий объём скачанных данных;
* рассчитывает среднюю скорость скачивания в МБ/с.

## Требования

* Python 3.10+
* pip

Проверить установленную версию Python:

```bash
python --version
```

или

```bash
python3 --version
```

## Конфигурация

Настройки приложения находятся в файле `config.json`.

Пример:

Cоздать файл `config.json` (пример файла в `config.json.example`)

```json
{
    "url": "https://<your domain>/100MB.bin",
    "requests_count": 10,
    "timeout": 60
}
```

### Параметры конфигурации

| Параметр         | Описание                             |
| ---------------- | ------------------------------------ |
| `url`            | URL файла для скачивания             |
| `requests_count` | Количество последовательных запросов |
| `timeout`        | Таймаут одного запроса в секундах    |

## Формула расчёта скорости

Скорость скачивания рассчитывается по формуле:

```text
скорость (МБ/с) =
    общий объем скачанных данных (МБ)
    ---------------------------------
    суммарное время скачивания (сек)
```

## Возможные ошибки

### Не найден файл конфигурации

```text
FileNotFoundError: Файл конфигурации 'config.json' не найден
```

Убедитесь, что файл `config.json` существует в корне проекта.

### Ошибка соединения

```text
requests.exceptions.ConnectionError
```

Проверьте:

* доступность указанного URL;
* наличие интернет-соединения;
* настройки прокси или VPN.
